# Tic-Tac-Toe Nodejs
Tic-Tac-Toe game 
TechStack Node.js, Socket.io , REPL

## Requirements
chalk: 4.1.2, <br>
express: 4.17.1, <br>
repl: 0.1.3,<br>
socket.io: 4.2.0,<br>
socket.io-client: 4.2.0 <br>

## Installation Notes
1.Clone Repository
2.npm install
3.Starting the server: a)cd src
                        b)node server.js 5050
4.Connecting client a)cd public/js
                     b)node client.js 127.0.0.1 5050
